OpenPass v0.2.1 发布，主要更新：

## 🚀 新功能

- 右键识别二维码（img/canvas/截屏兜底/站点回退）
- 2FA 多输入框分组填充与检测启发式及 UI 样式
- 密钥名称/站点点击跳转与表格长文本截断

## 🐛 Bug 修复

- pnpm allow builds commands
- 修复 pnpm install 的 ERR_PNPM_IGNORED_BUILDS
- secrets 自愈循环与构建警告，统一样式入口

## ♻️ 重构

- 拆分 App.vue 为组件并对齐 manager 设计系统
- 统一站点域名匹配到 domainMatch 模块

## 🔧 维护

- 移除残留的 package-lock.json，统一用 pnpm
- 移除手写 CHANGELOG.md，以 GitHub Releases 页面为唯一 changelog
- 迁移仓库链接至 kovawx 并接入发版脚本
